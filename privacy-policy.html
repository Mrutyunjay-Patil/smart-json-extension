Privacy Policy for Smart JSON

The 'Smart JSON' extension is designed to enhance the user experience when interacting with JSON data on web pages. It respects user privacy and operates entirely client-side, without collecting, storing, or transmitting any personal data. The extension processes JSON data present in the web pages visited by the user, but does not track, store, or transmit any of this data.

There is no collection of personally identifiable information, health information, financial details, authentication information, personal communications, location data, web history, user activity, or website content. As such, the extension does not require a privacy policy since it does not engage in any form of data collection or processing that impacts user privacy.
